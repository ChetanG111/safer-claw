# Introduction

Welcome to the documentation for your SaaS application.

## Prerequisites

- Bun runtime installed
- PostgreSQL database
- Node.js (optional, but recommended for some tools)

## Installation

1. Clone the repository
2. Run `bun install`
3. Setup `.env`
4. Run `bun dev`

Enjoy building!
