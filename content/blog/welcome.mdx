# Welcome to ShipFree!

We are excited to announce the launch of our new SaaS Starter Template.

## What is ShipFree?

ShipFree is a free, open-source boilerplate for building SaaS applications with Next.js, TailwindCSS, and Drizzle ORM.

### Key Features

- Authentication (Better-Auth)
- Payments (Stripe, Lemon Squeezy, Polar)
- Database (PostgreSQL with Drizzle)
- UI Components (Shadcn/Base UI style)

Stay tuned for more updates!
