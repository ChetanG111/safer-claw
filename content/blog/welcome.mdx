# Welcome to Safer-Claw!

We are excited to announce the launch of Safer-Claw.

## What is Safer-Claw?

Safer-Claw is a free, open-source boilerplate for building SaaS applications with Next.js, TailwindCSS, and Drizzle ORM.

### Key Features

- Authentication (Better-Auth)
- Payments (Stripe, Lemon Squeezy, Polar)
- Database (PostgreSQL with Drizzle)
- UI Components (Shadcn/Base UI style)

Stay tuned for more updates!
